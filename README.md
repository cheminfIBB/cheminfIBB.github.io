## Local Preview

1. Install ruby.

1. Install ```bundler``` with ```gem install bundler```.

1. Go to the repository and run ```bundle install```.

1. Run ```bundle exec jekyll serve --incremental```.

1. Go to ```localhost:4000``` in your browser.
